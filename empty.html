D、偽類別選擇器

1.連結類

a:link { background: aqua }
a:visited { background: pink }

範例：

2.使用者操作類

input:hover { background: aqua }
input:active { border-width: 5px; }
input:focus { background: yellow }

範例：

3.結構類(同層結構)

td:nth-child(3) { background: aqua; }
td:nth-last-child(3) { background: pink; }
td:first-child { background: red; }
td:last-child { background: blue; }
補充：
關鍵字：odd, even
公式：2n, 2n+1

範例：

4.結構類(同層並排除異元素)

td:nth-of-type(3) { background: aqua; }
td:nth-last-of-type(3) { background: pink; }
td:first-of-type { background: red; }
td:last-of-type { background: blue; }

範例：

5.UI狀態

input:disable { background: pink; }
input:checked { height: 30px; width: 30px; }

範例：

6.否定偽類

h2:not(.estela) { background: aqua; }
補充：
可以是元素, class, id, 屬性, 偽類等等...
